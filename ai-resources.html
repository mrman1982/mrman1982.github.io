General AI Assistants & Research Tools

  • Google Gemini — chat, multimodal understanding, and Docs/Slides help. Visit

    Great for brainstorming, rewriting, and images (in supported regions). Integrates with Google Workspace. Check data policies if using sensitive data.

  • OpenAI ChatGPT — versatile chat for writing, code, and Q&A. Visit

    Excellent general assistant. Use with care for private data; consider redaction or offline alternatives for sensitive content.

  • Microsoft Copilot — Microsoft’s assistant with Bing search. Visit

    Useful for quick answers, image generation, and Office integrations. Enterprise versions offer improved data controls.

  • Anthropic Claude — strong at long‑context reading and careful reasoning. Visit

    Good for summarising large documents and safer drafting. Review T&Cs for data handling if compliance is key.

  • Perplexity — research assistant with citations. Visit

    Ideal for quick research with source links. Helpful to validate facts before acting on them.

Image & Media Creation

  • Google Gemini (Images) — image generation and understanding. Visit

    Create images from prompts and understand images (availability varies by region/account).

  • OpenAI ChatGPT (Images) — image generation via integrated models. Visit

    Generate and iterate on images from natural language prompts alongside chat.

  • Adobe Firefly — generative images/text effects, integrated with Creative Cloud. Visit

    Great for marketing visuals and brand‑safe assets. Check licensing for commercial use.

  • Midjourney — high‑quality image generation (Discord‑based). Visit

    Fantastic artistic output. Useful for concept art and ideation.

  • Stable Diffusion — open‑source image models (local or cloud). Visit

    Flexible for custom styles and private setups. Good option for offline image workflows.

  • ComfyUI — node‑based workflows for image pipelines. Visit

    Visual graph editor to build and automate image generation/fine‑tuning workflows.

Local & Offline Options

  • ComfyUI (Local) — build private, reproducible image pipelines. Visit

    Run locally or on‑prem; great for controlled, auditable workflows.

  • Ollama — run LLMs locally with simple commands. Visit

    Handy for private chat and prototyping on your own machine or server.

  • LM Studio — desktop app to run local models. Visit

    Friendly UI for trying models without coding. Good for demos and small teams.

  • Stable Diffusion (Local) — image generation on‑prem. Visit

    Popular web UI options exist for private image pipelines and fine‑tuning.

  • FLUX (local models) — modern image model family. Visit

    Variants like FLUX.1 can run locally with the right setup; check model cards and licenses.

  • WAN 2.2 — community image model (niche use cases). Browse

    Explore on Hugging Face; review license and requirements before use.

Tips for Responsible Use

  • Never paste sensitive or regulated data into public tools; prefer offline or enterprise options.
  • Review licensing and usage rights for generated media, especially in commercial settings.
  • Validate facts with cited sources; treat outputs as drafts, not final truth.
